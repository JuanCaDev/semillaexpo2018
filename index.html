Carrera Tecnológica de la Ciencia y cambio climático

Para equipos de 3 estudiantes (1 debe ser estudiante de otra disciplina). Deben desarrollar un aplicativo superando diferentes desafíos sobre la Ciencia y Cambio Climático y de acuerdo a su puntuación se premiará los 3 primeros lugares.

Niveles de participación

Nivel 1

Construir diferentes modelos de un diagrama de flujo de datos para el cálculo de diferentes variables en el cambio climático

Nivel 2

Programación de aplicativo web con el uso de las siguientes alternativas: PHP, HTML, CSS, JavaScript, Framework (Bootstrap o Laravel) para el registro inicial de datos del cambio climático y manejo de información para la sensibilización de los ciudadanos.

Nivel 3

Programación de aplicativo de escritorio en Java que permita el procesamiento de las variables que se tienen en cuenta sobre el cambio climático.
